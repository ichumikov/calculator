Консольный калькулятор. Калькулятор поддерживает следующие действия: сложение, вычитание, умножение, деление, возведение в степень, расчет факториала

Для сборки проекта в директории проекта выполнить:

g++ src/app.cpp src/main.cpp src/math.cpp -o calculator

                  
Примеры команды для запуска:

./calculator 1 + 2
